# We Movies

# Instalation :

    1- git clone https://github.com/RabaouiSaber/movies.git

    2- docker-compose up --build

    3- docker exec -it symfony_app /bin/bash

    4- composer install

    5- npm install

    6- npm run dev

    7- ajouter le API_KEY de TheMovieDB dans le fichier .env

    8- accéder à 127.0.0.1:8080
